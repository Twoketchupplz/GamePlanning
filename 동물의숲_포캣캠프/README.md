### 개요
- 정의
  - 주요 콜렉션인 가구, 옷, 오브제를 제작하도록 한다.
  - 제작중인 물품을 확인하고 제작완료된 물품을 수집한다.
- 기획 의도
  - 유저가 직접 제작해 성취감을 느끼게한다.
  - 유저가 제작하길 원하는 수집품을 쉽고 빠르게 찾도록 한다.
  - 유저가 원치 않은 제작을 안하도록 보조한다.
  - 재료 재화를 소모시킨다.
- 시스템 구조
  - 전체 Flow chart
